# Original Implementation - Animal Shelter Dashboard

## Overview
This is the original implementation of the animal shelter dashboard developed for CS-340. The project demonstrates fundamental database operations and visualization capabilities through a web-based interface for managing animal shelter data.

## Core Components

### Database Operations
- MongoDB connection management
- Basic CRUD functionality
- Query implementation
- Data validation

### Web Dashboard
- Interactive filters for rescue types
- Data visualization components
- Geolocation mapping
- Dynamic data tables

### Visualization Features
- Map display of animal locations
- Pie chart for breed distribution
- Filtering capabilities
- Interactive data tables

## Requirements
- Python 3.x
- MongoDB
- Dash framework
- Plotly for visualization
